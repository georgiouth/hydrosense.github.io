How it works

A Pt electrode catalytically splits H2; the resulting protons de‑dope the organic semiconductor, producing an immediate drop in conductivity that we measure with ≤ 0.8 V bias.

The reversible p‑doping / de‑doping cycle allows thousands of repeat tests without drift.
